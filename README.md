Описание базы данных по ссылке https://edu.postgrespro.ru/bookings.pdf.
После того, как все entity были реализованы,
была проверена корректность (параметр hbm2ddl в update, чтобы hibernate сам построил таблицы).
После построения таблиц дамп, который находится в pdf по ссылке, заливается без ошибок
